# nkunamphobrian
My personal web portfolio site which showcases my work, skills and past experience within software engineering field. The site is currently built entirely on static framework using HTML, CSS &amp; JS.

## About this project

This project was scaffolded with AI assistance (Claude) to speed up initial 
build-out — layout, base styling, and starter logic. I directed the design 
and content decisions (structure, data model, feature set, copy) and have 
since been modifying the code directly: Added images.

I'm using AI tools the way I'd use any dev tool — to move faster on 
boilerplate — while focusing my own learning time on the logic, 
architecture, and features layered on top. Happy to walk through any part 
of the code or the decisions behind it.

Stack: HTML/CSS/vanilla JS, multiple-files static site.

Status: actively maintained.

What I built/changed myself: Changed generic details with my actual details including projects and their links, I added my resume allowing visitors to download it for job purposes.
